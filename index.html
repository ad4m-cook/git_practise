HTML Cheat Sheet

Elements and Stucture

Tag Name Description
<li> List Creates list elements inside of ordered and unordered lists
<ul> Unordered list Creates an unordered list
<ol> Ordered list Creates a numbered ordered list
<video> Video Embeds a media player for visual playback
<img> Image Creates an image
<a> Anchor Used to create hyperlinks
<h1>-<h6> Headings Six different levels of headings
<em> Emphasis element Emphasies text elements
<strong> Strong Makes the text bold
<br> Line break Creates a line break in a text element
<div> Div Creates a container that divides the HTML document into sections
<p> Paragraph Displays a block of text
<body> Body Represent the content of a HTML document
<head> Head Contains the metadata for a HTML document
<span> Span Used to group text elements for styling purposes
<title> <Title> <Displays the webpage's title in the browser>
<html> HTML declaration Declares the language being used in the document
<!--> Comment Creates a comment in the code that isn't interpreted as HTML


Attributes

Attribute Name Description
name="value" Attribute name Provides a name to a HTML attribute
id="value" Attribute ID Provides an ID to a HTML attribute
alt="text" Alt Provides an alternative description if an attribute fails to render
target="_blank" Target Opens attribute in a new tab in the browser
